To execute the Bartol_Brandon_Term_Project.py please change the value of path_1 to be the location of the shot data files.
The shot data files should be named as follows:
"NBA_2020_Shots.csv"
"NBA_2021_Shots.csv"
"NBA_2022_Shots.csv"
"NBA_2023_Shots.csv"
"NBA_2024_Shots.csv"

Please change the value of path_2 to the location of the NBA All-Star list.
This file should have a value: "NBA_ALL_STAR_BY_YEAR.csv"

Additionally the data files can be found: https://www.kaggle.com/datasets/mexwell/nba-shots/
